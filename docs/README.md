## 💡 참고사항

- 모든 코드는 **SOLID 원칙**을 준수합니다
- **Feature-Sliced Design** 아키텍처를 따릅니다
- 파일 크기는 **300줄 이내**로 제한됩니다
- **TypeScript strict 모드**를 사용합니다
